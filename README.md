**Focus-Timer**

A simple Pomodoro-style timer built with HTML, CSS, and JavaScript.

## Features

* **Focus / Break Timer:** Standard 25-minute focus sessions and 5-minute breaks.
* **Distraction Alert:** Sends a notification if you switch tabs during a focus session.
* **Idle Detection:** Pauses the timer if you're inactive for 2 minutes.
* **Notifications:** Uses browser notifications for alerts.

## How to Run

1.  Download `index.html`, `style.css`, and `script.js`.
2.  Put all three files in the same folder.
3.  Open `index.html` in your web browser.
4.  Allow notifications when prompted.
